# sprint7-final-project
Proyecto final Sprint 7
el objetivo del proyecto es entender cómo los clientes usan realmente los servicios móviles (llamadas y mensajes) de la empresa ConnectaTel.
Se utilizaron los datasets Plans, Users y Usage.
las etapas del análisis realizadas, primero se exploraron los datasests, posteriormente limpiaron y ajustaron para finalmenta analizarlos con la ayuda de gráficos y datos numéricos. 
cómo ejecutar el notebook; asegurate de tener instalada la versión de Python 3.7+.
una breve guía de reproducción. Ejecuta las celdas del notebook en orden secuencial para evitar errores de referencia de variables.
